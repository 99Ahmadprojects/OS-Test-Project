Home

This collaborative web project was created by our BSCS 5th Semester team as part of the Operating Systems course project. The purpose of this work is to practice teamwork and demonstrate how multiple members can collaborate on a single repository using Git. Our goal is to create a simple yet well-organized webpage that reflects each member’s contribution.

About Us

We are a team of BSCS students in the 5th Semester at Riphah International University, working together to complete our Operating Systems collaborative project. Each member is responsible for a specific part of the development process to ensure teamwork and learning through Git collaboration.

Team Members & Tasks

  • Sharaiz Ahmed [57288] – Responsible for creating the main HTML structure and layout of the webpage.
  • Shahid Zaman [55123] – Handling CSS design and styling to make the webpage visually appealing and responsive.
  • Mir Ahmed Shah [54906] – Implementing JavaScript features to add interactivity and functionality to the webpage.

Each team member contributes to their assigned part and commits changes to the shared Git repository. This ensures that all tasks are coordinated, and everyone’s work is clearly visible through version-control history.

Key Features

  • Responsive design for better user experience
  • Clean and semantic HTML structure
  • Stylish UI elements (to be added with CSS)
  • Interactive features (to be implemented with JavaScript)
  • Collaborative Git workflow for teamwork demonstration
