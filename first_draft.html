Welcome to Our Collaborative Web Project

This project was created by our BSCS 5th Semester team for the Operating Systems course. It's a demonstration of teamwork and collaborative development using Git, resulting in a simple yet well-organized webpage that reflects each member’s contribution.

About Us

We are a team of BSCS students from Riphah International University, working together on our Operating Systems collaborative project. Each member took on a specific role to practice teamwork and learn through Git collaboration.

Key Features

This project incorporates several key elements to ensure a quality outcome.

  • Responsive Design

    Ensures a great user experience on all devices, from mobile phones to desktops.

  • Clean & Semantic HTML

    A well-organized and meaningful structure that is good for SEO and accessibility.

  • Stylish UI Elements

    Modern design with attention to spacing, typography, and color palette.

  • Interactive Features

    Smooth scrolling, active navigation state, and mobile-friendly menu.
