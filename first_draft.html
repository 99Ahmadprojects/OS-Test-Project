Project Overview

This collaborative web project is created by our BSCS team pf 5th semester as part of our Operating Systems semester project. The purpose of this project is to practice teamwork and demonstrate how multiple members can work together on a single repository using Git.

Team Members & Tasks

  • Sharaiz Ahmed [57288] – Responsible for creating the main HTML structure and content layout.
  • Shahid Zaman [55123] – Handling CSS design and page styling to make the interface attractive and responsive.
  • Mir Ahmed Shah [54906] – Working on JavaScript functionalities to make the webpage interactive.

Each team member is contributing to their specific part and will commit changes to the shared Git repository. This ensures that all tasks are well-coordinated and everyone’s work is clearly visible through version control history.
